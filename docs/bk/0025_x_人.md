# 人

  - 人是由什么组成的？骨骼+血肉+思想？
  - 那么思想占多少？肯定是个大头，而且是可变的，从出生开始逐渐占的比例越来越大，然后止于或者趋于某个点(像是某种函数)。
  - 思想是飘渺的，现实才是看得见，摸得着的。当思想映射到行动，思想才能进一步得到升华。
  - 结果是一种状态，是现在也是过去，也是思想的沉淀与积累。
